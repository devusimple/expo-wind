# Avatar

## Overview
An avatar component that displays user initials on a colored background with configurable size.

## Installation
```bash
bunx rn-shadcn add avatar
# or
npx rn-shadcn add avatar
# or
pnpm rn-shadcn add avatar
```

## Import
```tsx
import { Avatar } from '@/components/ui/avatar'
```

## Usage
```tsx
<Avatar initials="JD" size="md" />
```
