# Badge

## Overview
A small badge component for displaying labels, statuses, or counts with multiple color variants.

## Installation
```bash
bunx rn-shadcn add badge
# or
npx rn-shadcn add badge
# or
pnpm rn-shadcn add badge
```

## Import
```tsx
import { Badge } from '@/components/ui/badge'
```

## Usage
```tsx
<Badge variant="default">New</Badge>
```
