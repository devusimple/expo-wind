# Label

## Overview
A label component for form fields that provides accessible text descriptions.

## Installation
```bash
bunx rn-shadcn add label
# or
npx rn-shadcn add label
# or
pnpm rn-shadcn add label
```

## Import
```tsx
import { Label } from '@/components/ui/label'
```

## Usage
```tsx
<Label nativeID="email-label">Email Address</Label>
```
