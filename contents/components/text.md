# Text

## Overview
A typography component with predefined style variants for consistent text rendering across the app.

## Installation
```bash
bunx rn-shadcn add text
# or
npx rn-shadcn add text
# or
pnpm rn-shadcn add text
```

## Import
```tsx
import { Text } from '@/components/ui/text'
```

## Usage
```tsx
<Text variant="h1">Heading 1</Text>
<Text variant="p">Paragraph text with default styling.</Text>
<Text variant="muted">Muted text for less emphasis.</Text>
```
