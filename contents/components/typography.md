# Typography

## Overview
A set of typography components for consistent heading, paragraph, list, and inline text styles.

## Installation
```bash
bunx rn-shadcn add typography
# or
npx rn-shadcn add typography
# or
pnpm rn-shadcn add typography
```

## Import
```tsx
import { H1, H2, H3, H4, P, Blockquote, Code, Lead, Large, Small, Muted } from '@/components/ui/typography'
```

## Usage
```tsx
<H1>Heading 1</H1>
<H2>Heading 2</H2>
<P>Paragraph text with proper line height and spacing.</P>
<Blockquote>Blockquote for emphasized quotes.</Blockquote>
<Code>Inline code</Code>
```
